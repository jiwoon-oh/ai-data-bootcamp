# 전처리 파이프 라인 품질 리포트
## 전처리 단계(pipe방식 사용)
1. 결측치 및 중복행 제거
    - 결측치(amount/customer_age) 제거: -87행 → 1413
    - 나이 범위(10~80) 밖 제거: -8행 → 1405
    - quantity > 10 제거: -1행 → 1404
    - 중복행 제거: -0행 → 1404
        => 최종: 1500 → 1404 (총 96행 제거, 6.4%)
2. 분석용 파생 칼럼 생성
    - memership 칼럼에서 등급에 따라 차등을 두기 위해 Ordinal encoding 진행
    - region, channel, category 칼럼에서는 순서가 상관없으므로 One-hot encoding 진행
        => 최종: (1404, 9) → (1404, 12)으로 파생 칼럼 3개 추가
3. 스케일링
    - 스케일링 단계에서 결측치나 이상치가 많은것으로 의심되어 Robust scaling 진행
        => 최종: (1404, 12) → (1404, 25) 인코딩 칼럼 13개 추가
4. 로그 생성
    - 각 단계별 제거와 생성 숫자를 파악하기 위해 로그 함수 생성

=> 전처리 후 데이터: (1404, 28)

## 전처리 후 데이터(상세)
- 전체 데이터: (1500,9) -> (1404,28)
- 제거된 비율: 6.4%
- 칼럼별 결측 수: "amount": 60, "customer_age": 30
- 전처리 후 데이터 결측 없음
